translator

Translator sends data from Courier to the Frontend, interacting with Database as necessary

Stream translated story data

  • Requires user_id, story_id, step, and chapter_number as query parameters.
  • Returns a text/event-stream of translation chunks.
  • Errors out if any required field is missing or invalid.
query Parameters
user_id
required
string

The ID of the user requesting translation.

story_id
required
string

The ID of the story being processed.

step
required
string
Enum: "generate_outline" "critique_outline" "rewrite_outline" "generate_first_chapter" "generate_next_chapter" "critique_chapter" "rewrite_chapter"

Which story step to translate.

chapter_number
required
integer >= 0

Chapter index (0-based or 1-based, per your convention).

Responses

https://capstone-projects-2025-spring.github.io/translate
