# Backend API
            
#### import ApiSchema from '@theme/ApiSchema';

#### <ApiSchema pointer="./backend.yaml" />;

openapi: 3.0.0
```
info:
  title: Story Builder API
  version: 0.0.0
  description: API for communication between frontend user interface, LLM Agents, and MongoDB Database.
```
